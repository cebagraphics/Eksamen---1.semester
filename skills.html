Jeg elsker at lære nye ting og forbedre mig. Herunder kan du se hvilke kompetencer jeg har i værktøjskassen på det givne tidspunkt, som måske kan bidrage til jeres virksomhed.

I Adobe InDesign kan jeg skabe designs og layouts til både trykte og digitale udgivelser.
I Adobe After Effects kan jeg skabe bevægelsesdesign (animationer) og visuelle effekter.
I Adobe Photoshop kan jeg redigere billeder, skabe grafikker/illustrationer og lave mockups.
I Adobe Lightroom kan jeg redigere billeder så de bliver præcis som man vil have dem.
I Adobe Premiere Pro kan jeg redigere enhver type video, så det passer til det givne projekt.
I VS Code kan jeg kode simpel html og css.
I Figma kan jeg skabe wireframes og prototyper til apps, hjemmesider m.m.
